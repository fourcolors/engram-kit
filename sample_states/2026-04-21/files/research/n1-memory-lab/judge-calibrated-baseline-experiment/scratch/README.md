# Scratch Convention

Scratch files in this directory are allowed to be temporary, but every artifact
should be reproducible enough for a future session to understand why it exists.

Use this naming pattern:

```text
YYYYMMDD_short_description.ext
```

Recommended header for scripts:

```text
Purpose:
Inputs:
Outputs:
Known caveats:
```

Do not treat scratch outputs as claims until they are summarized in
`RUN_LOG.md` or promoted into `EXPERIMENT_CHECKLIST.md`.
